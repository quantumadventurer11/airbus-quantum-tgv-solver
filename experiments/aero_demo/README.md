# Aerodynamic Demo Experiments

These files are exploratory classical CFD visualizations and are not part of
the official Airbus Taylor-Green Vortex challenge submission. They are retained
only as optional background experiments.

## Contents

- `generate_aero_visuals.py` — Classical D2Q9 LBM simulation of a NACA 0012
  airfoil (Re=300) and a cylinder wake / Kármán vortex street (Re=150).
- `figures/` — Output images and animations from those runs.

## Note

The official Airbus challenge submission addresses only the **2D Convecting
Taylor-Green Vortex** benchmark. See the project root `README.md` and
`reports/final_report.md` for the challenge-scoped work.
